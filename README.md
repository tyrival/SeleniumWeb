# Selenium Web
## 概述
Selenium Web是Selenium Manager的前端源码。采用Vue+Element开发。

**前端源码**：[Selenium Web](https://github.com/tyrival/SeleniumWeb)

**后端源码**：[Selenium Manager](https://github.com/tyrival/SeleniumManager)




## 命令

- npm install

  安装依赖包

- npm run dev

  开发模式

- npm run build

  编译

- npm run server

  测试
